# Homebrew Formulae for Tinted Theming

## [Tinty]

```shell
brew tap tinted-theming/tinted
brew install tinty
```

## [builder-rust]

```shell
brew tap tinted-theming/tinted
brew install builder-rust
```

[Tinty]: https://github.com/tinted-theming/tinty
[builder-rust]: https://github.com/tinted-theming/builder-rust
